What is Docker?

  • Just another layer of abstraction
  • Some tool for automation of OS-level virtualization
  • A hype ... it will pass over
  • Some virtual machine

But it ...

  • Is clean, small, lightweight, safe, portable and fast
  • Summarizes software into images
  • Encapsulates and isolates dependencies
  • Is easy to scale, reuse and distribute

Docker in a Nutshell

  • Based on linux containers (isolated processes but shared kernel)
  • Instructions are read from a Dockerfile (plain text with commands)
  • Docker can assemble containers bases on these files
  • Runtime images of a container are called images

Examples

  • Show all images
 docker images
  • List of containers
 docker ps -a
  • Reuse
 docker start –-attach container name
  • Stop
 docker stop container name
  • Remove
 docker rm container name
  • ...
 ....

Examples

  • Alpine linux: docker run -it alpine sh
  • Nginx: docker run --name docker-nginx -p 80:80 nginx -d
  • There is an image for everything you need
