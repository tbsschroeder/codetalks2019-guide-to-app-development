What is Docker?

Docker is just ...
  • another layer of abstraction
  • another tool for automation of OS-level virtualization
  • a hype ... it will pass over
  • some virtual machine

But it ...

  • is small, lightweight and fast
  • summarizes software into images
  • encaspulates and isolates dependencies
  • is easy to scale, reuse and distribute

Docker in a Nutshell

  • Instructions are read from a Dockerfile
  • Dockerfiles are plain text with commands
  • Docker can assemble containers bases on these files
  • Runtime images of a container are called images

Examples

  • Show all images with docker images
  • List of containers docker ps -a
  • Reuse with docker start –attach
  • Stop with docker stop
  • Remove with docker rm
  • ...

Examples

  • Alpine linux: docker run -it alpine sh
  • Nginx: docker run --name docker-nginx -p 80:80 nginx -d
