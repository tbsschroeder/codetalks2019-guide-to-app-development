Code Reviews with GitLab

How to enable merge request approvals

Basics

  • Settings > Repository > Protect a Branch
  • Use the git-push.sh script

git-push.sh


#!/bin/bash

read -p "User Story: " story
read -p "Author:" author
echo "Pushing a review on story $story by $author. Thank you\!"

branch=$(git branch | grep \* | cut -d ' ' -f2)
head=$(git rev-parse --short HEAD)

echo "Move unpushed commits to new branch"
git branch $branch"_"$story"_"$author"_"$head
git checkout $branch"_"$story"_"$author"_"$head

echo "Push new branch"
git push --set-upstream origin  $branch"_"$story"_"$author"_"$head

echo "Go to old branch"
git checkout $branch
git branch -D $branch"_"$story"_"$author"_"$head

echo "Please visit:"
echo "https://git.metrosystems.net/order-management/betty_ordermanagement_returns/merge_requests/new?merge_request%5Bsource_branch%5D="$branch"_"$story"_"$author"_"$head"&merge_request%5Btarget_branch%5D=master"
echo " to open the merge request and assign one of your colleagues."
echo ""
echo "Have a nice day\!"
